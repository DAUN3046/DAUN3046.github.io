# 문제발생
Git bash에서 경로를 설정할 때 다음과 같은 오류가 발생한다.
```
bash: cd: too many arguments
```
# 발생원인
경로 폴더명에 공백이 들어가 있는 부분이 문제였다/.

# 문제해결
공백이 들어간 폴더명을 쌍따옴표(`""`)로 묶어서 적으면 잘 설정된다.
